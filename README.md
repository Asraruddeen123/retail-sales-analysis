# 🛒 Retail Sales Performance & Customer Insights Analysis

## 📌 Objective
Analyze 4 years of retail sales data to uncover revenue drivers, 
loss-making segments, and actionable business insights.

## 🛠️ Tools Used
| Tool | Purpose |
|------|---------|
| Excel | Data cleaning, Pivot Tables, EDA |
| SQL (SQLite) | Business queries, aggregations, segmentation |
| Python | Exploratory Data Analysis, visualizations |
| Power BI | Interactive executive dashboard |

## 📊 Key Findings
- **Technology** is the highest revenue category
- **Tables** and **Bookcases** are loss-making sub-categories
- Higher discounts strongly correlate with lower profit
- **West region** leads in sales
- **Consumer segment** drives the most revenue
